# Contact

Use this page to find the official contact options for Breezell support, product questions, and documentation feedback.

## Email

For questions, support requests, or documentation feedback, contact Breezell by email:

[email@breezell.com](mailto:email@breezell.com)

When sending a message, include a clear description of your request, any relevant configuration details, and the steps you have already tried. This helps the team review your question more efficiently.

::: tip
We will reply to your email within 16–24 hours.
:::

## Website

Visit the official Breezell website for product information, resources, and the latest updates:

## Documentation Feedback

If you find an issue in Breezell Docs or notice content that could be clearer, please include the page name, the section title, and a short explanation of the problem when contacting us.
